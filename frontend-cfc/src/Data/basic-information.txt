Roles in organization
Core team : handles all other teams and members
each region has : 
    Executive body (EB)  : upto 13 members
    College Representative (CR)  : can be any number 
    General Member (GM)  : can be any number but most no. of cr * 3 (3 per cr)

Tech lead maintains pages and websites is thus considered as admin, core team is superadmin, all other members have no special permissions in the website and pages. 

Certificate is generated by tech leads , can be generated by superadmin but mostly done by tech leads.

Super admin works : add, remove admins (tech leads)
admin (tech lead ) works : add/remove/verify members (gm, cr, eb), add/update/delete posts/blogs/jobs/events/gallery, generate certificates,
