# Code for Change - Official Website

![Code for Change](/frontend-cfc/public/logo.png)

Welcome to the **Official Website Repository** for **Code for Change (CFC)**, a non-profit organization dedicated to empowering IT students across Nepal.

This platform bridges the gap between academia and the tech industry, offering resources such as CSIT/BCA notes, tech events (Codefest, Tech Conclave), mentorship programs, and internships.

---

## About Code for Change Nepal

**Code for Change** aims to cultivate a thriving tech community by providing students with the guidance, resources, and platforms they need to excel in the IT sector.

- **Events:** Hosting major tech events like _Codefest_ and _Tech Conclave_.
- **Resources:** Providing structured study materials (CSIT, BCA) for academic success.
- **Community:** Facilitating a space for students, mentors, college representatives, and industry experts to connect.

## Getting Started (Frontend)

To run the frontend application locally, follow these steps:

1. **Navigate to the frontend directory:**

   ```bash
   cd frontend-cfc
   ```

2. **Install dependencies:**
   Make sure you have Node.js and `npm` or `pnpm` installed.

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Open in Browser:**
   The application will typically run on `http://localhost:5173`.
